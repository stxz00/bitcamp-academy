LIST

내가 작성한 바로 이전 문장을 보여줌

SQL> list
  1  select
  2  *
  3  from
  4* dept
------------------------------------------------------------------
Run또는 / 또는 R: 이전 명령을 실행함(재실행)

ED or EDIT :  수행할 내용을 메모장에 적고 저장 

START 메모장이름 또는 @ 메모장이름  : 저장했던 메모장 내용 수행

edit ex01.sql
내용 넣고 저장 후
start ex01.sql

SET HEADING OFF 또는 ON :  출력제어

SET HEADING OFF 후
SQL> select * from dept where deptno=30;
하면 테이블이 안나오고 로우값만 나옴

SET HEADING ON 후
select * from dept where deptno=30;

형식을 지정하기 위한 명령어
칼럼 제목을 성정하기 위한 COULUM HEADING
column ename; 하면 정의한 적이 없음
column ename format a7 (알파벳 사이즈로 7개사이즈까지만 ___표시)

column sal format 0,000 (숫자를 0,000 형식으로 해줌, 단 0,800,형식)
column sal format 9,999 하면 (800으로 정상으로 출력)
------------------------------------------------------------------
column sal 하면 칼럼 sal의 형식 나옴
column deptno format 00.00  소수점 형식

단, 특정 테이블 안의 deptno 칼럼이라고 하지 않아 dept쪽의 deptno도 형식이 바뀜
